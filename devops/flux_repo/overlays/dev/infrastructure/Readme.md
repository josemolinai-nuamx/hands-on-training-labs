# ⚙️ Infraestructura Dev

Define la infraestructura específica del entorno **Dev**, derivada de las bases comunes.

## Componentes
- **prometheus/** → Stack de monitoreo adaptado a desarrollo.
- **reposources/** → Repositorios Helm requeridos para las apps Dev.
- **vault-besu/** → Repositorios para HashicorpVault.

> ✨ Ideal para validar integraciones antes de promover cambios a staging o producción.
