# 📚 Helm Repositories (Base)

Contiene la lista de **HelmRepository** utilizados por FluxCD para instalar charts desde distintas fuentes.

## Repositorios Incluidos
- Appscode
- Bitnami
- Grafana
- Jetstack
- Hashicorp
- Prometheus Community
- CloudNativePG
- Ingress Nginx
- JFrog, Harbor, etc.

> 📦 Estos recursos son sincronizados automáticamente por FluxCD para permitir despliegues HelmRelease desde múltiples orígenes.
